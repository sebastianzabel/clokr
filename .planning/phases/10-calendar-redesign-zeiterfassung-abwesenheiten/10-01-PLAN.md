<objective>
Redesign the Zeiterfassung monthly calendar (`time-entries/+page.svelte`) from a shared-border grid to a gap-based island grid. Each calendar cell becomes a visually distinct surface with rounded corners, breathing room between cells, and a colored left-border stripe communicating work status. The legend is cleaned up to float inline without a gray background bar. All hardcoded hex values in calendar-related styles are replaced with CSS custom property equivalents for proper dark-theme support.

Purpose: Match the Clockodeo-inspired visual language — airy cell surfaces, subtle depth, status communicated via color accent, not color flood.
Output: Modified scoped `<style>` block in `time-entries/+page.svelte`. No markup changes, no logic changes, no API changes. One change to `app.css` to raise `.cal-other` opacity from 0.30 to 0.35 (D-12 — shared global rule, affects both pages).
</objective>

**2a. Add colored left-border stripe to status cells (D-05):**
The status cell rules currently only set `background`. Add `border-left: 3px solid` to each. Final rules (lines ~1554-1568):

```css
.cal-cell--ok {
  background: var(--color-green-bg);
  border-left: 3px solid var(--color-green);
}
.cal-cell--partial {
  background: var(--color-yellow-bg);

  background: var(--color-red-bg);
  border-left: 3px solid var(--color-red);

  background: var(--color-green-bg);
  border-left: 3px solid var(--color-green);
}
.cal-cell--today-partial {
  background: var(--color-yellow-bg);
  border-left: 3px solid var(--color-yellow);
}
```

Note: The `border-left` here is a semantic status stripe, not a grid border — it works correctly on gap-based island cells. The global `.cal-abs-*` rules use `!important` on background only and do not declare `border-left`, so absence cells with a status class will show the stripe alongside the absence background. This is acceptable — research (Risk 3) confirmed status is `none` for most absence days, so stripe classes will not be applied in practice.

**2b. Redesign `.cal-legend` — remove gray background and border-top (D-10, D-11):**

```css
  flex-wrap: wrap;
}
```
Removed: `border-top`, `background`. The legend now floats cleanly as a pill row below the grid, matching the pill-style design intent. Gap reduced from 1.25rem to 1rem for tighter pill feel.
