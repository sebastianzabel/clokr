**1d. Fix `.cal-cell--drag-selected` (lines ~2678-2682) — CRITICAL BUG FIX:**
The current `outline` does not follow `border-radius`. With rounded cells (6px), the outline renders as a rectangle creating a visual mismatch. Replace `outline` + `outline-offset` with `inset box-shadow` (same pattern as `.cal-today` in app.css):

Replace:
```css
.cal-cell--drag-selected {
  background: var(--color-brand-tint, rgba(109, 40, 217, 0.1)) !important;
  outline: 2px solid var(--color-brand, #6d28d9);
  outline-offset: -2px;
}
```
With:
```css
.cal-cell--drag-selected {
  background: var(--color-brand-tint, rgba(109, 40, 217, 0.1)) !important;
  box-shadow: inset 0 0 0 2px var(--color-brand);
}
```
The `inset box-shadow` respects `border-radius` and creates a ring identical in visual weight to the old outline, but now following the rounded cell shape. This matches the pattern used by `.cal-today` in `app.css` (line 1661).
