<objective>
Self-host Google Fonts locally to comply with DSGVO Art. 44 (no external CDN requests to Google).

Purpose: Every page load currently requests fonts from fonts.googleapis.com and fonts.gstatic.com, transmitting user IP addresses to Google servers. This violates DSGVO Art. 44 (transfer of personal data to third countries without adequate protection). Per D-07 through D-10, all font files must be downloaded locally and served from the app's own static directory. GitHub Issue #100.

Output: 8 WOFF2 font files in static/fonts/, updated app.css with local @font-face, updated hooks.server.ts CSP without Google domains.
</objective>
