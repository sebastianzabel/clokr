<objective>
Add a Playwright `storageState` setup project that authenticates once and reuses the session across all E2E specs.

Purpose: Every E2E test currently calls `loginAsAdmin()` at the start, wasting 5-10 seconds per spec. The storageState pattern logs in once per test run and shares the session across all browser projects. This is the canonical Playwright auth pattern.
Output: `auth.setup.ts` saves `.auth/admin.json`; playwright.config.ts wires setup project with `dependencies: ["setup"]` on all browser projects.
</objective>
